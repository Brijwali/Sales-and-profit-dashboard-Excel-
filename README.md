# Sales-and-profit-dashboard-Excel-
This dashboard analyzes Sales and Profit data using Excel Pivot Tables, Pivot Charts, and slicers to track overall performance and trends. It highlights key KPIs such as total sales, total profit, and profitability across different segments, enabling quick, data-driven insights through interactive filtering and clean visual design.

Overview:
This project is an interactive Excel dashboard built using Pivot Tables,
Pivot Charts, slicers, and custom formatting to analyze sales performance.

Key Features:
- KPI cards (Revenue, Profit, Growth %)
- Dynamic Pivot Tables with slicers
- GETPIVOTDATA for stable references
- Custom number formatting ($, K, M)
- Clean dashboard layout

Tools Used:
- Microsoft Excel
- Pivot Tables & Charts
- Advanced formulas

How to Use:
1. Download the Excel file
2. Enable editing
3. Use slicers to interact with the dashboard
